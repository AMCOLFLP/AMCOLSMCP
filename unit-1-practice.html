Exercise 1A: Marker Selection Fill-in-the-Blanks

Type the best answer in each blank, then press Check Answers.

1. ________. Alter course to starboard. Hint: Order that must be followed.
2. ________. You are running into danger. Hint: Danger marker.
3. ________. I will reduce speed. Hint: Planned action marker.
4. ________. What are your intentions? Hint: Asking marker.
5. ________. I require immediate assistance. Hint: Asking for help marker.

Expanded Activities, Quizzes, and Knowledge Checks

How to use the interactive activities

Complete the exercises in sequence: vocabulary recognition first, grammar control second, marker decision third, and role-play production last. For Unit 1, students should repeat the role-play aloud after typing it, because SMCP is a speaking skill as well as a writing skill.

Teacher Note: These added activities provide extra assessment after the lesson. Students should complete the quiz first, then the decision check, and finally the scenario writing challenge.

Exercise 7D: Knowledge Check Quiz

Choose the best answer for each item. These questions test understanding, not memorization only.

1. Which marker is used for a required action or command?
2. Which marker is used for a recommended safe action?
3. Which marker is used when stating a planned maneuver?
4. Which marker is used to ask for assistance or service?
5. Which marker is best before “What are your intentions?”

Exercise 7E: Marker and Procedure Decision Check

Select the most accurate SMCP marker, priority signal, or operational decision for each situation.

1. “Reduce speed to six knots.”
2. “You are running into shallow water.”
3. “My main engine is disabled.”
4. “I require tug assistance.”
5. “I will pass astern of you.”

Student Self-Assessment Checklist

Using this checklist honestly

Students should tick an item only when they can perform it without looking at the model answer. If a student cannot explain why a marker is correct, the item should remain unticked and the student should review the grammar and sample conversation again.

Before moving to the next unit, tick the skills you can perform confidently.
