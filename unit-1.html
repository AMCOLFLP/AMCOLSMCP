Expanded Main Discussion: markers as safety labels

Unit 1 ties the whole module together. A message marker is not decoration; it tells the listener how to interpret the sentence before the details arrive. This is essential when radio quality is poor, accents differ, or the situation is developing quickly.

Students should practise choosing markers by asking one question: “What is the purpose of my message?” If the purpose is to order, use Instruction. If the purpose is to warn, use Warning. If the purpose is to state future action, use Intention. This unit should be used repeatedly as a reference during all other units.

Focus: This unit helps students understand, choose, and use all eight IMO SMCP message markers in real maritime communication. The purpose is to make spoken messages clear, short, and safe during routine operations, port arrival, emergency response, collision avoidance, onboard work, and navigational warnings.

Student-Friendly Key Idea: A message marker tells the listener the purpose of your message before the details are spoken. It is like a safety label for communication. When the marker is correct, the listener immediately knows whether the message is an Instruction, Advice, Warning, Information, Question, Answer, Request, or Intention.

Learning Outcomes

By the end of this unit, students should be able to:

  • Define and distinguish all eight SMCP message markers.
  • Choose the correct marker based on communication purpose.
  • Use the correct grammar pattern after each marker.
  • Correct common marker mistakes in VHF and onboard communication.
  • Produce an integrated maritime conversation using all eight markers accurately.
Why this matters on board: Message markers are the core safety labels of SMCP communication. When students master them, their speech becomes clearer, shorter, and safer across all maritime situations.

Course Position Note: This unit is now placed first because the eight message markers are the foundation used in every later SMCP situation. Students should master the purpose of each marker before moving to routine VHF, emergency messages, pilotage, collision avoidance, mooring, and warnings.
