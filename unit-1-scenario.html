Core Feature: The Message Marker Decision System

In real operations, a speaker has only a few seconds to choose the correct marker. A practical system is to identify the function of the message first, then choose the marker, then speak the details. This prevents mixed messages such as giving an order in the form of a casual question or reporting danger without clearly saying that it is a warning.

1. Identify the situation: routine report, danger, order, request, planned action, or question.

2. Choose one marker before speaking the main message.

3. Use short SMCP wording after the marker.

4. Avoid slang, jokes, local expressions, and long explanations on VHF.

5. For critical information, require read-back or confirmation.

6. Correct misunderstanding immediately using “Say again,” “Correction,” or a clear Question.

Common Student Mistake: Students often use Information for every message because it feels safe. However, “Information. Reduce speed” is incorrect because “Reduce speed” is an order. Use Instruction. Reduce speed. Another common mistake is using Question when actually making a Request. “Question. Give me a tug” is incorrect. Use Request. I require tug assistance.

Interactive Exercises

All-marker production checklist

  1. Use Instruction only for required actions.
  2. Use Advice for recommended safer action.
  3. Use Warning before danger.
  4. Use Information for facts and status.
  5. Use Question to ask for information.
  6. Use Answer to reply or read back.
  7. Use Request for assistance, permission, or service.
  8. Use Intention for planned future action.

Sample Scenario Section: Sample Scenario: Choosing All 8 SMCP Message Markers in One Complex Operation

Student Task: Read the operational scenario carefully. Then answer the comprehensive questions. Your answers should connect vocabulary, grammar, SMCP markers, operational procedure, and correct maritime phraseology from this unit.

MV Ocean Legacy is approaching a busy port in reduced visibility. VTS reports a disabled fishing vessel near the fairway, two outbound container ships, and one dredger restricted in her ability to manoeuvre. The pilot station asks for the vessel's draught, ETA, and tug requirements. The master plans to reduce speed, keep to the starboard side of the fairway, and request two tugs for berthing. At the same time, VTS recommends extra caution and later orders Ocean Legacy to report at Waypoint Delta.

During the approach, the OOW sees a small target crossing ahead on radar. The bearing is steady, and the range is decreasing. The master wants to warn the small craft, ask its intentions, state Ocean Legacy's own intention, and report the situation to VTS. The bridge team must use all eight SMCP message markers correctly: Instruction, Advice, Warning, Information, Question, Answer, Request, and Intention.

Students must decide which marker fits each communication purpose. The challenge is not only to know the marker names, but to select the safest marker under operational pressure.

Teacher Notes / Expected Answer Points
  • This unit tests marker selection more than vocabulary memory.
  • Students should show the communication purpose before the details.
  • Full-credit conversations should include all eight markers and use concise maritime wording.

How to answer the scenario questions

For this unit scenario, students should not copy long sentences from the passage. A strong answer should identify the operational problem, choose the correct SMCP marker, use the target grammar from Unit 1, and produce a safe standard phrase. Teachers can ask students to justify why they selected each marker.
