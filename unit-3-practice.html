Exercise 3A: Distress, Urgency, and Safety Fill-in-the-Blanks

Type the best answer in each blank, then press Check Answers.

1. A vessel in grave and imminent danger sends a ________ message. Hint: Distress priority.
2. A vessel with engine failure but no immediate danger sends a ________ message. Hint: Urgency priority.
3. A navigational warning to all ships begins with ________. Hint: Safety priority.
4. ________. I require immediate assistance. Hint: Marker for asking help.
5. ________. I am sinking in position Latitude one-three degrees North. Hint: Marker for facts/status.

Expanded Activities, Quizzes, and Knowledge Checks

How to use the interactive activities

Complete the exercises in sequence: vocabulary recognition first, grammar control second, marker decision third, and role-play production last. For Unit 3, students should repeat the role-play aloud after typing it, because SMCP is a speaking skill as well as a writing skill.

Teacher Note: These added activities provide extra assessment after the lesson. Students should complete the quiz first, then the decision check, and finally the scenario writing challenge.

Exercise 2D: Knowledge Check Quiz

Choose the best answer for each item. These questions test understanding, not memorization only.

1. Which priority signal is used for grave and imminent danger to life or vessel?
2. Which signal is most suitable for engine failure when there is no immediate danger?
3. A drifting container warning to all ships should normally begin with...
4. Which information is essential in a distress message?
5. What equipment transmits distress position to search and rescue systems?

Exercise 2E: Marker and Procedure Decision Check

Select the most accurate SMCP marker, priority signal, or operational decision for each situation.

1. Uncontrolled fire in the engine room with danger to life.
2. Main engine failure, vessel drifting, but no immediate danger.
3. Coast radio broadcasts a storm warning to all vessels.
4. You say: “I require immediate assistance.”
5. You say: “Twenty-two persons on board.”

Student Self-Assessment Checklist

Using this checklist honestly

Students should tick an item only when they can perform it without looking at the model answer. If a student cannot explain why a marker is correct, the item should remain unticked and the student should review the grammar and sample conversation again.

Before moving to the next unit, tick the skills you can perform confidently.

Unit Exit Ticket

Decide whether each situation needs MAYDAY, PAN-PAN, or SECURITE: engine failure near fairway, drifting container, uncontrolled engine-room fire, injured crew member, unlit buoy.
