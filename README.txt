Maritime English SMCP Modern UI Course Website

Open index.html in a browser.

This modernized version includes:
- Cleaner responsive interface with modern cards, dashboard, grouped navigation, and course map.
- Unit 1 as the foundations of all eight SMCP message markers.
- Combined assessments page for pre-test and post-test.
- Audio-player placeholders for unit conversations and listening/pronunciation activities.
- Sidebar search, active page highlighting, mobile menu, floating back-to-top and course-map buttons.
- Offline-ready standalone HTML/CSS/JavaScript.

Audio files should be placed in the assets folder using the filenames listed in the course materials.

Navigation update: The top navigation is now compact and grouped into Course, Units, and Practice dropdown menus.
