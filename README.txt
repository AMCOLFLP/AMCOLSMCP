Maritime English SMCP Multi-Page Interactive Website

Open index.html to start. Keep all files and the assets folder in the same directory.
Each unit is on a separate page:
- unit-1.html to unit-7.html
- resources.html

Interactive features:
- Fill-in-the-blank instant checking
- Matching dropdown instant checking
- Role-play keyword/marker analysis
- Unit completion progress saved in the browser
- Print / Save as PDF button
- Page search
