FINAL NAVIGATION IMPROVEMENTS
- The Course Navigator button now reliably shows and hides the left directory.
- The chosen Navigator state is remembered by the browser.
- Every learning unit has a compact quick-navigation bar with:
  • Previous and next unit buttons
  • A Unit selector
  • An “On this page” section selector
  • A Course Navigator show/hide button
- Headings use smooth section jumps and remain visible below the sticky top bar.

THAI ASSIST
- Thai Assist remains OFF by default.
- Turn it on from the top bar.
- Click the compact “ไทย” button beside selected discussions for a detailed Thai pop-up.

ROLE ACCESS
- Administrator: full access plus browser-based lesson editing.
- Teacher: full course, assessment, answer-key, rubric, and resource access without admin editing.
- Student: pre-test, Units 1–7, practice, resources, and capstone. The post-test unlocks after all seven units are completed.

SECURITY NOTE
This is an offline classroom prototype. Public institutional deployment requires secure server-side authentication, a database, and server-side authorization.
