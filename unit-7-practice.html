Exercise 7A: Weather and Navigational Warning Fill-in-the-Blanks

Type the best answer in each blank, then press Check Answers.

1. ________. Visibility restricted by heavy rain. Hint: Marker for danger.
2. Wind ________ to south-west means the wind changes counter-clockwise. Hint: Opposite of veering.
3. A sudden sharp increase in wind speed is called a ________. Hint: Weather term.
4. SECURITE messages often include navigational or ________ warnings. Hint: Weather-related.
5. Advice. Keep a sharp ________ for drifting hazards. Hint: Standard safety phrase.

Expanded Activities, Quizzes, and Knowledge Checks

How to use the interactive activities

Complete the exercises in sequence: vocabulary recognition first, grammar control second, marker decision third, and role-play production last. For Unit 7, students should repeat the role-play aloud after typing it, because SMCP is a speaking skill as well as a writing skill.

Teacher Note: These added activities provide extra assessment after the lesson. Students should complete the quiz first, then the decision check, and finally the scenario writing challenge.

Exercise 6D: Knowledge Check Quiz

3. “Visibility restricted by fog” describes...
4. An unlit buoy is an example of...

Exercise 6E: Marker and Procedure Decision Check

Select the most accurate SMCP marker, priority signal, or operational decision for each situation.

1. A drifting container is reported in the traffic lane.
2. You report: “Wind veering north-west Force eight.”
3. Coast radio says: “Keep clear of the area.”
4. You ask if the reported hazard is inside the traffic lane.
5. VTS requires all vessels to report when passing the hazard.

Student Self-Assessment Checklist

Using this checklist honestly

Students should tick an item only when they can perform it without looking at the model answer. If a student cannot explain why a marker is correct, the item should remain unticked and the student should review the grammar and sample conversation again.

Before moving to the next unit, tick the skills you can perform confidently.
